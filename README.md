# ConJugBasq

Sistema que gestiona la información de los jugadores y equipos de la NBA, usando los servicios de la API https://free-nba.p.rapidapi.com/. De igual manera tiene un filtro, ordenamiento, paginado, también se trabaja con los formularios de manera temporal en relación a la data. Debe registarse en la página de la api, y generar su apiKey, y cambiarla en los environments
